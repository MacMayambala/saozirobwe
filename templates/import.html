Excel Format Instructions

Ensure your Excel file includes columns with the following headers:

  • Required fields: branch, mem_reg_date, member_number, first_name, surname, gender, dob, id_number, card_number, phone_number1, kin_sname, kin_fname, kin_phone
  • Date formats: Use YYYY-MM-DD for mem_reg_date and dob
  • Phone numbers: 12 digits starting with 256 (e.g., 256123456789)
  • ID number: Pattern CF or CM + 11 characters + letter (e.g., CF12345678901A)
